1 ⬇️

安装桌面应用

下载并安装 Claw Wallet 桌面应用。你的私钥将存储在这里，不会与 AI 共享。

立即下载 →
2 🤖

安装 Skill

用一条命令为 AI Agent 添加 Claw Wallet 能力。支持 Claude Code、OpenClaw、Cursor、Cline。

npx skills add janespace-ai/claw-wallet
3 🔗

生成配对码

在桌面应用中点击「配对」→「生成配对码」，将 8 位配对码发送给你的 Agent 即可完成配对。

4 ✨

开始使用

让 Agent 转账、在 Uniswap 兑换，或与任何 DeFi 协议交互。每笔交易都需要你在桌面端审批。

安全优先的架构设计

三组件隔离架构，端对端加密。即便 AI 模型完全被攻破，攻击者也无法获得任何密钥。 

AI Agent        中继服务器          桌面钱包
（无密钥）  ←──E2EE──►  （密文转发）  ←──E2EE──►  （持有所有密钥）

X25519 + AES-256-GCM 端对端加密
中继服务器只转发密文，无法读取或篡改任何消息
AI Agent

只能看到钱包地址和交易哈希。不持有任何密钥。

中继服务器

无状态 WebSocket 转发器。只收到密文，无法读取或修改消息内容。

桌面钱包

所有密钥存储于此，本地签名。每次操作都展示审批界面。

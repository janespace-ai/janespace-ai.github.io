几分钟快速上手

四步完成配置，让你的 AI Agent 拥有真正的链上钱包。

1 ⬇️

安装桌面应用

下载并安装 Claw Wallet 桌面应用。你的私钥将存储在这里，不会与 AI 共享。

macOS → Windows →
2 🤖

安装 Skill

用一条命令为 AI Agent 添加 Claw Wallet 能力。支持 Claude Code、OpenClaw、Cursor、Cline。

使用 OpenClaw？直接在对话框告诉它：

openclaw plugins install @janespace-ai/claw-wallet

使用 Claude Code、Cline、Cursor 或其他 Agent？粘贴到 Agent 对话框：

Install Claw Wallet: https://github.com/janespace-ai/claw-wallet

或通过 CLI 安装：

npx skills add janespace-ai/claw-wallet
3 🔗

生成配对码

在桌面应用中点击「配对」→「生成配对码」，将 8 位配对码发送给你的 Agent 即可完成配对。

4 ✨

开始使用

让 Agent 转账、在 Uniswap 兑换，或与任何 DeFi 协议交互。每笔交易都需要你在桌面端审批。

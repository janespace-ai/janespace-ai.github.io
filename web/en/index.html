Get started in minutes

Four steps to give your AI Agent a real on-chain wallet.

1 ⬇️

Install Desktop App

Download and install the Claw Wallet desktop app. Your private keys will be stored here, never shared with the AI.

Download →
2 🤖

Install the Skill

Add the Claw Wallet skill to your AI agent with one command. Works with Claude Code, OpenClaw, Cursor, and Cline.

npx skills add janespace-ai/claw-wallet

Or paste directly into your agent chat:

Install Claw Wallet: https://github.com/janespace-ai/claw-wallet
3 🔗

Generate Pairing Code

In the desktop app, tap Pair → Generate Code. Share the 8-character code with your agent to link them together.

4 ✨

Start Using

Ask your agent to send tokens, swap on Uniswap, or interact with any DeFi protocol. You approve every transaction on desktop.

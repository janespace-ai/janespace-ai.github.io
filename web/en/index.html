Get started in minutes

Four steps to give your AI Agent a real on-chain wallet.

1 ⬇️

Install Desktop App

Download and install the Claw Wallet desktop app. Your private keys will be stored here, never shared with the AI.

Download →
2 🤖

Install the Skill

Add the Claw Wallet skill to your AI agent with one command. Works with Claude Code, OpenClaw, Cursor, and Cline.

npx skills add janespace-ai/claw-wallet
3 🔗

Generate Pairing Code

In the desktop app, tap Pair → Generate Code. Share the 8-character code with your agent to link them together.

4 ✨

Start Using

Ask your agent to send tokens, swap on Uniswap, or interact with any DeFi protocol. You approve every transaction on desktop.

Security-first architecture

Three isolated components. End-to-end encrypted. Even if the AI model is fully compromised, the attacker gains zero key material. 

AI Agent          Relay Server        Desktop Wallet
(zero keys)  ←──E2EE──►  (ciphertext)  ←──E2EE──►  (holds all keys)

X25519 + AES-256-GCM end-to-end encryption
The relay only forwards ciphertext — it can never read messages
AI Agent

Sees wallet addresses and transaction hashes only. Zero secrets.

Relay Server

Stateless WebSocket forwarder. Receives only ciphertext, cannot read or modify messages.

Desktop Wallet

All keys live here. Signs locally. Shows approval UI for every request.
